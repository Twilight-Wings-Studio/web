プライバシーポリシー / Privacy Policy

最終更新日：2025年10月15日
Last Updated: October 15, 2025

1. アプリケーションについて / About the Application

本アプリケーション（以下「本アプリ」）は、YouTubeのライブ配信コメントを取得・表示するためのツールです。
ユーザー自身のYouTubeアカウントを使用してログインし、視聴中のライブ配信に関するコメント情報を表示します。

This application (“the App”) is a tool for retrieving and displaying live chat messages from YouTube Live broadcasts.
Users log in using their own YouTube account to view comments from live streams they are watching.

2. 取得する情報 / Information We Collect

本アプリは、Google OAuth 2.0 認証を通じて、次の情報にアクセスする場合があります：

  • YouTubeアカウントの基本情報（ユーザー名、アイコンURL）

  • ライブ配信コメント（テキスト、投稿時刻、コメント投稿者情報）

  • ライブチャットに関連するメタデータ（バッジ、エモート、課金メッセージ等）

These data are accessed through Google OAuth 2.0 authentication and include:

  • Basic YouTube account information (display name, profile image URL)

  • Live chat messages (text content, timestamp, user info)

  • Related metadata (badges, emotes, paid messages)

※この情報は、YouTube Data API v3 のスコープ
https://www.googleapis.com/auth/youtube.readonly
の範囲内でのみ使用されます。

Note: This data access is strictly limited to the YouTube Data API v3 scope
https://www.googleapis.com/auth/youtube.readonly.

3. 情報の利用目的 / Purpose of Data Use

取得したデータは以下の目的のみに使用されます：

  • ユーザーが視聴しているライブ配信のコメントを画面上に表示するため

  • アプリの機能提供および動作確認のため

The collected information is used only for:

  • Displaying live chat messages for the live broadcast being viewed

  • Providing and maintaining the app’s functionality

4. 情報の保存 / Data Storage

  • 認証トークン（アクセストークン・リフレッシュトークン）は、ユーザーの端末上でのみ安全に暗号化されて保存されます。

  • 当方（開発者）はサーバー側でこれらのデータを保持・収集・転送しません。

Authentication tokens are securely stored only on the user’s local device, encrypted by the application.
No user data or tokens are collected, stored, or transmitted to any external server controlled by the developer.

5. 情報の共有 / Data Sharing

  • 本アプリは、第三者に個人情報を共有または販売することはありません。

  • Google以外の外部サービスへのデータ送信は一切行いません。

This app does not share or sell user information to any third party.
No data is transmitted to external services other than Google APIs.

6. ユーザーによるデータ削除 / Data Deletion by User

ユーザーは、以下の方法で自らの認証情報を削除できます：

  • アプリ内の「連携解除」機能を使用する

  • Googleアカウントの「セキュリティ設定 → アカウントへのアクセスを許可したアプリ」ページで、本アプリへのアクセス権を取り消す

Users can revoke and delete their authentication data at any time by:

  • Using the “Unlink” or “Disconnect” feature within the app

  • Visiting the Google Account settings page: https://myaccount.google.com/permissions

7. データの第三者送信 / Third-Party Transfers

本アプリは、ユーザーのデータを開発者サーバーまたは他の第三者へ送信しません。
Google API との通信はすべて HTTPS により暗号化されています。

This app does not transmit user data to any developer-owned or third-party server.
All communication with Google APIs is encrypted via HTTPS.
